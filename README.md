# Work_in_practic

Прототип платформы live-streaming e-commerce с аукционом в реальном времени.

## Возможности

- **Прямой эфир** — стример запускает трансляцию с камеры (телефона/компьютера), зрители видят видео в реальном времени
- **Чат** — зрители общаются без регистрации, выбирают имя и пишут сообщения
- **Live-аукцион** — ставки на товар в режиме реального времени, обновление цены
- **Карточка товара** — несколько вариантов (Кофта 1, 2, 3), переключение по миниатюрам, кнопка BUY NOW

## Запуск

```bash
./run.sh
```

Скрипт создаст виртуальное окружение, установит зависимости и запустит сервер.

Требуется Python 3.

## URL

- **Зрители:** `http://localhost:50260/stream`
- **Стример:** `http://localhost:50260/start_stream`

Для доступа с телефона используйте IP вашего компьютера в локальной сети (например, `http://192.168.1.XXX:50260/stream`).
